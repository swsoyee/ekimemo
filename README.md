### 確認したバグ

1. 日本語の「黒」字の表示がエラーを出る可能性あり、しばらくデータベースにある「黒」を全て中国語の「黑」にした。
2. DataTableのfilterが日本語に対してエラーを得られる、更に「運」も使えない。
